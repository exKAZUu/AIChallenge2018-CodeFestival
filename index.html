# AI Challenge とは

AI Challenge 2018 は CODE FESTIVAL に併設されたゲーム AI プログラミングコンテストです！

ゲーム AI プログラミングコンテストは主催者が提供するゲーム（例えば囲碁や将棋など）に対して、 参加者がコンピュータプレイヤーを操作する AI プログラムを作ることで競い合うコンテストです。 ゲームをプレイする感覚でプログラミングを楽しめるため、初学者向けの教材として注目を浴びています。

本コンテストでは、将棋や囲碁よりもシンプルで同じくらい奥深いルールのゲームを提供しています。 プログラミングを始めたばかりのあなたでも楽しめる内容となっていますので、是非ご参加ください！
